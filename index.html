    Início de carreira

    Minha carreira teve início como programador técnico em eletrônica, no desenvolvimento de software para automação industrial, programando CLP, IHM, Inversores de frequência e drivers de servo motores em uma empresa que fabrica máquinas industriais. Utilizei ferramentas de diversos fabricantes, como Rockwell, Schneider, Siemens, Elipse, Weg entre outros. Como linguagens de programação, tenho experiência com STL(Structure Text Language), Ladder, FBD (Function Block Diagram) e Java. Com formação técnica em mecatrônica através do SENAI, sempre tive contato com linguagens de programação como C, C++ e C# (CSharp).

    2024

    Conhecimentos técnicos

    Iniciei minha carreira como programador com formação técnica em mecatrônica, desenvolvendo software para automação industrial. Em paralelo, busquei aprimoramento profissional no desenvolvimento de software com ASP.NET e Windows Forms. Fiz formações e certificações através da plataforma Alura e Desenvolvedor.io buscando através da prática constante um maior aprimoramento. Hoje, minha especialidade em desenvolvimento é no ambiente .NET utilizando a linguagem C# (CSharp). Um resumo de meus conhecimentos tecnológicos: Protocolos de comunicação, SqlServer, MySql, RabbitMQ, CSharp, Docker, .NET Core, ASP.NET, Windows Forms.

    2024

    Atualmente

    Atualmente estou trabalhando como desenvolvedor de software para soluções de controle e gerenciamento de terminais de combustíveis, utilizando a metodologia de gerenciamento de projetos Scrum. Desenvolvi uma solução para parametrização à distância de computadores de vazão, com protocolo TCP/IP Proprietário. No momento, atuo no desenvolvendo uma solução de agendamento e em paralelo uma API que será consumida por uma aplicação mobile para motoristas. Principais tecnologias utilizadas nestes desenvolvimentos: Ambiente .NET, Windows Forms, .NET Core, ASP.NET, Visual Studio, Azure DevOps, Docker, RabbitMQ, processos de CI/CD.
